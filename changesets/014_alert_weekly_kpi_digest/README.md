# 014_alert_weekly_kpi_digest

Weekly Mon 8 AM PHT cron. KPI digest: star leakage, pass rate, drop-off, source ROI.

| Kind | xml_id |
|---|---|
| Server action | `bicc_recruitment.weekly_kpi_digest` |
| Cron | `bicc_recruitment.cron_weekly_kpi_digest` — every 1 week, nextcall `2026-05-18 00:00:00` (= next Mon 8 AM PHT) |

## QA on dev

Run the SA manually. HR Recruitment Manager group inbox gets a digest like:
> 1. Star Leakage  
> 2. Overall Pipeline  
> 3. Assessment Drop-off (Jeepney Test)  
> 4. Source ROI
